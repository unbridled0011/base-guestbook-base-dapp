# Deployment Notes

This public repository intentionally omits private deployment secrets.

Use `.env.example` as the template and keep real values only in local `.env.local` or Vercel environment variables.

Never commit private keys, seed phrases, RPC keys, API keys, GitHub tokens, or Vercel tokens.
