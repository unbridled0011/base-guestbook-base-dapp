# Base Guestbook

## App Name

Base Guestbook

## Tagline

Leave your Base note

## Description

Write a short message on Base and sign a public guestbook that updates live with onchain entries.

Character count: 95

## Category

Social / community / onchain message wall

## Short Pitch

Base Guestbook gives people a fast way to leave a public note on Base, turning a simple wallet interaction into a living message wall.

## Production Environment Variables

- `NEXT_PUBLIC_GUESTBOOK_CONTRACT_ADDRESS`: deployed guestbook contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
